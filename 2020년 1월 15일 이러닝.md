```python
def calc_sum(*params):
    total=0
    for val in params:
        total += val
    return total
ret_val=calc_sum(1,2)
ret_val=calc_sum(1,2,3)
```

#### 클로져

중첩함수에서 중첩함수를 포함하는 함수의 스코프에서 접근 가능

```python
def outer_fn():
	id=0
	
	def inner_func():
		nonlocal id
		id+=1
		return id
	return inner_func()
```

주의할 것!! outer_fn()이 반환하는 값은 inner_func()의 주소값이다. 따라서 함수를 호출된 결과는 함수가 호출되는 것이다.

```python

def area(r):
    return 3.14*r*r
def circum(r):
    return 2*3.14*r
area_val=area(5)
circum_val=circum(5)

print("원 면적: {0}, 원 둘레: {1}".format(area_val, circum_val))
```
